# bdi-frontend


Steps to run the code:
Frontend -
To run the Frontend code:
1.	git clone https://github.com/Adil515139/bdi-frontend.git
2.	npm install
3.	Run “ng serve”
4.	Application will run on your localhost e.g. -  http://localhost:4200/

Library/Tools/Framework:
1.	Angular 8.0.0
2.	Node 18.17.1
3.	npm- 9.6.7
